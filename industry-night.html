Industry Night

Description:

This project was a group undertaking, where each member had their own role to play, creating a responsive page. This page is designed for the Fanshawe College Industry Night Event, where people come to view Fanshawe student's portfolio. This site has some placeholder content in it, and is still a work in progress, but it shows a great deal of the work of everyone involved. The site contains information about the event, as well as containing info about students.

Partners:

I worked on this project with three of my classmates: Dev Pandya, Jeet Patel and Sahilkumar Gosaliya. I helped with a lot of front end, working on HTML and CSS as well as some of the responsive design and a small amount of JavaScript. Dev worked on the promotional video, as well as other info about the event, as well as assisting with front-end design. Jeet did a lot of the design work for the site, and helped with implementing it on the front-end. Sahilkumar did a lot of front-end work, as well as doing revisions to design, and contributing a lot of the Javascript.

Skills Improved:

Wireframing - I worked on some of the inital low-fidelity designs for the site, though a lot of that was revised and improved by the others; not much of the initial layout made it to the current version of the site. Wireframing was done using Adobe XD.

Front End Development - I worked on some of the front-end work for this site, using HTML, CSS and a little Javascript.

Collaboration - This was my first project where I made a site as a member of a group of four. We all have a similar set of skills, but excel in our own areas. The challenge was to collaborate effectively, and try to break down the project into chunks, and determining who would work on what. We collaborated using GitHub.
